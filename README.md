## âš™ï¸ Technical Approach

Since only STL was available, the following geometric features were extracted using `numpy-stl`:

- Volume  
- Bounding box dimensions (X, Y, Z)  
- Surface area  
- Triangle count  
- Mean triangle area  
- Standard deviation of triangle area  
- Min/Max triangle area  
- Aspect ratio  
- Compactness  

These features were then used to train a regression model to predict machining manhours.

---

## ğŸ§  Feature Extraction Concept

From each STL mesh:

- Reshape triangle vertices

- Compute triangle areas via cross product  
- Derive geometric complexity indicators  

The goal was to approximate machining difficulty from:

- Material removal volume  
- Toolpath complexity  
- Surface machining demand  
- Geometric irregularity
